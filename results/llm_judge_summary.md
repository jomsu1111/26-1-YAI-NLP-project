# LLM as Judge 실험 결과 요약

## 실험 설정

- **모델**: gpt-5-mini
- **방법**: 도메인별 1,000개 stratified sampling (label 비율 유지) × 3 도메인 = 3,000개
- **태스크**: context + question + response를 보고 환각 여부 binary 판정 (0=정상, 1=환각)

---

## 전체 성능

| Metric    | Score  |
|-----------|--------|
| Accuracy  | 0.7258 |
| F1        | 0.7592 |
| Precision | 0.6782 |
| Recall    | 0.8623 |

- 환각 Recall이 0.86으로 높아 **환각을 놓치는 경우가 적음**
- 정상 샘플을 환각으로 오판하는 경우(615건)가 많아 **Precision은 상대적으로 낮음**

---

## Confusion Matrix

## 도메인별 성능

| 도메인        | n     | Accuracy | F1     | Precision | Recall |
|--------------|-------|----------|--------|-----------|--------|
| qa           | 1,000 | 0.8680   | 0.8556 | 0.9467    | 0.7804 |
| summarization| 998   | 0.6713   | 0.7153 | 0.6329    | 0.8224 |

- **qa**가 Accuracy 0.87로 가장 좋은 성능
- **dialogue**는 Recall 0.98로 환각을 거의 다 잡지만 Precision 0.58로 오탐이 많음
- **summarization**은 중간 수준
